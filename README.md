# Social Talks

Social Talks is a modern discussion web application where users can:

- Create opinion posts
- Agree or disagree (one vote per user)
- Reply to posts
- Reply to replies (nested discussions)
- Like replies
- Edit account details

## Features

- User Registration & Login
- Account Profile Page
- Edit Profile Option
- Single Vote System
- Nested Reply System
- Like System on Replies
- Clean Modern UI (Dark Theme)

## Tech Stack

- React.js
- React Router
- LocalStorage (Frontend Authentication)

## How to Run

1. Clone the repository
2. Run `npm install`
3. Run `npm start`
